SIMULATION
Dieser Ordner enthält Modelle, Parameter und Szenarien zur Simulation des Societal OS.
Simulationen helfen dabei, die Abgabe‑Mechanik, Renditepfade, Infrastrukturzyklen und Kapitalpuffer quantitativ zu testen.

Mögliche Inhalte:

Parameter für Abgabensätze (10–20 %)

Renditepfade für Infrastrukturinvestitionen

Modelle für Bürgerfonds und Reservefonds

Szenarien für Krisen, Demografie und Drift

Python‑Skripte zur Berechnung von Kapitalflüssen

Simulationen dienen der Kalibrierung und Validierung der Architektur.
